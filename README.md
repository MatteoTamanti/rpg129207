# 📌 BERK

A 2D RPG game developed in Java using JavaFX.  
The player explores a dungeon environment, interacts with an NPC, and faces enemies while progressing through different rooms.

---

## 🚀 How to Run the Project

### Requirements
- Java 21 (LTS)
- Gradle

### Instructions

```bash
git clone https://github.com/MatteoTamanti/rpg129207
cd rpg129207
```

### Project build
```bash
./gradlew build
```

### Running
```bash
./gradlew run
```

## 🤖 Use of AI tools

During the development of this project, ChatGPT and Claude Code were used as programming assistants. AI tools were used mainly as a support resource for debugging, clarifying specific technical concepts, and improving code quality.

The main activities performed with the support of AI were:

- **Debugging:** assistance in identifying possible causes of compilation errors, runtime issues, and unexpected behaviors during development.
- **Concept clarification:** support in understanding specific technologies and concepts introduced in the project, such as JavaFX and data persistence.
- **Technical support:** assistance in evaluating possible solutions to specific implementation problems encountered during development.
- **Code quality improvement:** suggestions for improving code readability, organization, and maintainability.

AI tools were used only as a support resource. The project architecture, design choices, and implementation of the main features were developed personally. Any suggestions provided by AI were reviewed, understood, and adapted before being integrated into the final code.
